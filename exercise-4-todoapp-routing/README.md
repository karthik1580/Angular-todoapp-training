# Todoapp

This project is adding routing to the TODO app so you can route to a completed todos page.
The steps done are:

1) Add routes and routing module forRoot in a app.routes file with routes for root and completed TODOs
2) Import appRoutes in AppModule
3) Add functionality to complete TODOs
4) Create component to filter completed TODOs
